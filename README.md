# Finding-Shortest-Paths-by-Genetic-Algorithm

## For What 

I am interested in Optimazation Problem, Especially Shortest Path Problem.

[Traveling Salesman Problem](https://en.wikipedia.org/wiki/Travelling_salesman_problem)
